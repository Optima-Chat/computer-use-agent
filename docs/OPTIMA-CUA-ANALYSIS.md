# Optima x Computer Use Agent 机会分析

**日期**: 2026-03-09

---

## Optima 背景

Optima AI 是一个 AI 原生的对话式电商平台，帮助中小商户和个人创作者 5 分钟开店、AI 自动化运营。核心定位是"AI 商业顾问"而非纯工具。

- **技术栈**: TypeScript + React + Next.js (前端), Python + FastAPI (后端), 60+ MCP 工具
- **目标客户**: 独立创作者、社交电商商户、深圳批发商、平台商户（Amazon/Temu 迁移）
- **现状**: MVP 已上线，3-4 商户测试中，$1.5M 融资已承诺，runway 3-6 个月

---

## 机会方向

### 1. 竞品平台操作自动化（一键搬家）

商户从 Amazon/Temu/Shopify 迁移到 Optima 时，Computer Use Agent 可以：
- 自动登录原平台，抓取商品信息（标题、图片、价格、描述）
- 操作原平台 UI 批量导出数据
- 降低迁移摩擦，提高获客转化率

**价值**: 获客利器，竞品难复制。对"平台商户寻求独立"这个客群（10-20M）尤其有吸引力。

### 2. 无 API 服务的兜底集成

Optima 已有 60+ MCP 工具覆盖主流服务，但总有些场景没有 API：
- 部分物流商后台（只有网页界面）
- 报关/清关系统
- 小众支付平台
- 政府/行业平台

Computer Use 可以作为万能胶水层："有 API 用 API，没 API 用 computer use"。

### 3. 竞品分析自动化

Agent 自动浏览竞品网站，截图分析：
- 页面设计和产品布局
- 定价策略
- 促销活动
- 生成结构化竞品报告

与 Optima "AI 商业顾问"定位天然契合。

### 4. 商户实时演示/培训

Optima 强调"教商户理解生意"而非纯代劳。Computer Use 可以：
- 在商户屏幕上实时操作演示
- 边做边教，比文档/视频更直观
- 增强"AI 导师"的产品体验

## 现实约束

| 约束 | 说明 |
|------|------|
| **成本** | 单次复杂 GUI 任务 ~$1-5，高频使用成本不低 |
| **稳定性** | OSWorld 最高 75%（GPT-5.4），生产环境还不够可靠 |
| **优先级** | Optima 当前应聚焦融资和 PMF，不宜分散精力 |
| **MCP 已够用** | 60+ 工具已覆盖大部分有 API 的场景 |
| **技术成熟度** | 赛道快速进化中，6-12 个月后成本和性能会显著改善 |

## 建议时间线

```
┌─────────────────────────────────────────────────────────┐
│                    Optima CUA 路线图                      │
├──────────┬──────────────────┬────────────────────────────┤
│  短期     │     中期          │         长期               │
│ (现在)    │  (6-12 个月)      │      (12+ 个月)            │
├──────────┼──────────────────┼────────────────────────────┤
│ 不投入    │ 探索"一键搬家"    │ MCP 工具链兜底层            │
│ 聚焦融资  │ 作为差异化功能     │ "有 API 用 API，            │
│ 和 PMF   │ 降低获客摩擦       │  没 API 用 computer use"   │
│          │                  │                            │
│ 持续关注  │ 竞品分析自动化     │ 实时演示/培训功能           │
│ 技术进展  │ 作为顾问功能增强   │ 全自动化运营助手            │
└──────────┴──────────────────┴────────────────────────────┘
```

### 短期（现在）
- 不需要投入开发资源
- 保持对 computer use 技术的跟踪（本 repo 的 benchmark 已搭好）
- 聚焦核心产品和 PMF

### 中期（6-12 个月）
- 探索"一键搬家"功能，作为获客差异化
- 竞品分析自动化，增强 AI 顾问能力
- 预计 token 成本和模型性能都会进一步改善

### 长期（12+ 个月）
- Computer Use 作为 MCP 工具链的兜底层
- 实时演示/培训功能
- 如果成本降到足够低，可以成为全自动化运营的基础能力

## 与 Computer Use Agent 市场的关系

详见 [MARKET-RESEARCH.md](MARKET-RESEARCH.md)

Optima 更可能是 computer use 技术的**应用方**而非**平台方**。竞争对手是 Shopify/Genstore，不是 OpenAI/Anthropic。关注的应该是：
- 哪个 CUA 平台最适合集成（Anthropic API vs OpenAI CUA vs 开源 Browser Use）
- 成本何时降到可接受水平
- 稳定性何时达到生产级别
